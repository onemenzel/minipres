📚 Complete Feature Reference

Let's explore all the features we've built into this framework!

🏗️ Core Architecture

  • Custom Elements: slide-deck, slide-page, slide-viewport
  • 4:3 Aspect Ratio: Fixed 1600×1200px slides that scale proportionally
  • Absolute Positioning: Slides removed from document flow
  • Smart Scaling: ResizeObserver + CSS custom properties
  • Self-contained: Single HTML file with no external dependencies

🎨 Slide Transitions

Global & Local Control

  • Set on slide-deck: transition="fade"
  • Separate in/out: transition-in="slide" transition-out="zoom-out"
  • Override per slide: <slide-page transition="slide">

Available Types

  • fade: Opacity transition
  • slide: Horizontal movement
  • slide-rotate: Slide + Y-axis rotation
  • zoom: Scale in from small
  • zoom-out: Scale out to large

Proper z-index layering ensures smooth animations!

✨ Progressive Element Animations

Element-Level Transitions

Add transition="..." to any element:

  • transition="fade" - Simple opacity
  • transition="slide-up" - Slides from below
  • transition="slide-left" - Slides from right
  • transition="zoom-in" - Scales in

Headlines always show immediately, other elements animate progressively!

📱 Responsive Design

Mobile Portrait

  • Reduced margin: 24px → 48px

  • Better space utilization

Mobile Landscape

  • Flex row layout

🖼️ Smart Image Loading

Optimized performance with lazy loading

  • Adjacent preloading: Current + previous + next slides
  • Lazy by default: Use loading="lazy" on images
  • JS override: Changes to loading="eager" when needed
  • Custom positioning: Works despite our absolute slide positioning

Perfect for presentations with many high-res images!
